    Global Objects

    GameState

    The main global game object containing all the game play information.

    GameState.roomIdstring

    The game room id.

    GameState.ownerIdstring

    The id of owner of the current game room.

    GameState.stateint

    The game current state.

    valueconstantdescription
    -1STATE_GAME_NOT_STARTEDgame as not started
    0STATE_GAME_IN_PROGRESSgame is in progress
    1STATE_GAME_OVERgame is over

    GameState.tickint

    The current tick. Reset each time the state of the GameState.state changes.

    GameState.tilesstring

    The stage of the game represented as a string. Each character represent a tile. For the dimension of the stage, see width and height

    valueconstantdescriptionimage
    .TILES_EMPTYempty tile
    +TILES_BLOCKblock tile
    #TILES_PILLARpillar tile

    GameState.playersarray<string, Player>

    A hash containing all the players with the player id as hash keys.

    GameState.bombsarray<string, Bomb>

    A hash containing all the bombs with the bomb id as hash keys.

    GameState.bonusesarray<string, Bonus>

    A hash containing all the bonuses with the bonus id as hash keys.

    GameState.explosionsstring

    A string that indicated the x:y coordinates of the explosions.

    GameState.widthint

    The width of the stage.

    GameState.heightint

    The height of the stage.

    GameState.tickDurationint

    The tick duration in ms. This is the game speed.

    GameState.suddenDeathCountdownint

    The countdown until sudden death occurs. For more information on sudden death, see Game mechanics.

    GameState.suddenDeathEnabledbool

    Whether the game is in sudden death or not. For more information on sudden death, see Game mechanics.

    GameState.isSimulationPausedbool

    Whether the game is paused or not.

    Player

    The player object containing all the player information.

    idstring

    The player id.

    namestring

    The name of the player.

    connectedbool

    Whether the player is connected or not.

    xint

    The x coordinate of the position of that player.

    yint

    The y coordinate of the position of that player.

    bombsLeftint

    The number of bombs the player can still drop.

    maxBombsint

    The maximum number of bombs the player can drop.

    Can be increased by picking up a Bonus.

    bombRangeint

    The range of the player bombs explosion.

    Can be increased by picking up a Bonus.

    alivebool

    Whether the player is alive or dead.

    respawningint

    A countdown indicating that the player is respawning. This happens after a player dies (see Game mechanics)

    scoreint

    The score of the player. For more information, see Game mechanics

    colorint

    The color of the player. This is an HEX in decimal.

    hasWonbool

    Whether the player as won or not.

    Bomb

    The bomb object containing all the bomb information.

    idstring

    The player id.

    Bonus

    The bonus object containing all the bonus information.
